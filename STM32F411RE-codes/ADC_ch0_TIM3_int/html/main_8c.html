Detailed Description

Exemplo uso ADC disparado com TIM3 e debouce de botao ADC é configurado para ser disparado pelo evento de TIM3 TIM3 é configurado para gerar um evento a cada update Interrupcao de TIM3 realiza o debounce do botao TIM10 é configurado para gerar interrupcao periodica Interrupcao aciona o led e dispara envio do valor convertido pela serial.
